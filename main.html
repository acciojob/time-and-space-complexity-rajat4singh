Understanding Time and Space Complexity

Time complexity of an algorithm quantifies the amount of time taken by an algorithm to run as a function of the length of the input. Note that the time to run is a function of the length of the input and not the actual execution time of the machine on which the algorithm is running on. It is efficient measure in algorithm analysis.

The amount of memory required by the algorithm to solve given problem is called Space Complexity of the algorithm. If we need to create an array of size n, this will require O(n) space.
